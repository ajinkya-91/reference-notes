So if you have solved the previous challenge of dolor and dolore, you might have found out how /dolor/ not only find's "dolor" but also highlights "dolor in word "dolore". So to fix this we simply add whitespace at the end.

So to find "dolor" regex is /dolor /

And to find "dolore" regex is /dolore /

123-456, 425-958, 426-545, 24584, 4564486, 958-542, 452875, 44656, 58635, 854-452, 958-452
